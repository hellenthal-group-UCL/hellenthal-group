Chromopainterv2

Chromopainterv2 is a piece of software which can be used to infer patterns of haplotype sharing between individuals/populations. This information can be used to, amongst other things, infer population structure (fineSTRUCTURE), date admixture events (GLOBETROTTER) and estimate admixture proportions (SOURCEFIND). The original paper which describes the technical details can be found here here, and the source code, examples and instructions can be downloaded here:
https://github.com/sahwa/ChromoPainterv2

SOURCEFIND

SOURCEFIND is an R package which can be used to infer copying proportions from the output of ChromoPainter and can be downloaded from:
https://github.com/sahwa/sourcefindV2


The technical description of the algorithm can be found in the supplementary methods of Latin Americans show wide-spread Converso ancestry and imprint of local Native ancestry on physical appearance.

GLOBETROTTER

GLOBETROTTER is piece of software which is able to infer and date admixture events, in addition to estimating admixture proportions from ChromoPainter output and can be downloaded from:
https://github.com/sahwa/GLOBETROTTER


The technical description of the algorithm can be found in the supplementary methods of A genetic atlas of human admixture history.
